# 🔑 Gestión de Personal
Controla quién tiene acceso al sistema y qué funciones puede realizar.

## Roles del Sistema
- **Administrador:** Tiene acceso total, incluyendo Reportes y Gestión de Empleados.
- **Usuario Estándar:** Solo tiene acceso a Ventas y Consultas.

## Seguridad y Contraseñas
- Al registrar un empleado, se debe definir un **Login** y **Contraseña**.
- **MD5:** Por seguridad, las contraseñas se encriptan. Nadie puede ver la contraseña original una vez guardada.
- Si un empleado olvida su clave, un administrador deberá actualizarla desde este módulo.

## Actualización
Para cambiar el rol de un empleado, búscalo en la lista, selecciona el nuevo puesto y presiona **Actualizar**.
