# 🌮 Catálogo de Productos
En este módulo puedes gestionar todos los alimentos y bebidas que se ofrecen en **Tacos Gómez**.

## Registro de nuevos productos
1. Haz clic en el botón **Nuevo**. El sistema asignará automáticamente el siguiente ID disponible.
2. Ingresa el **Nombre del Producto** (ej. Tacos de Pastor, Gringa, Refresco 600ml).
3. Define el **Precio ($)**. Puedes usar los controles de flecha para ajustar el costo.
4. Selecciona una **Categoría**:
   * **Tacos:** Para órdenes individuales o por orden.
   * **Bebidas:** Refrescos, aguas naturales y jugos.
   * **Platillos:** Especialidades de la casa.
5. Haz clic en **Grabar** para guardar en la base de datos.

## Modificación de Precios
Si necesitas actualizar el costo de un producto:
- Selecciónalo en la lista central.
- Cambia el valor en el campo **Precio ($)**.
- Presiona **Modificar**. El cambio se reflejará inmediatamente en el módulo de Ventas.

> **Nota:** El sistema utiliza el formato de moneda local configurado en Windows para mostrar los precios.
