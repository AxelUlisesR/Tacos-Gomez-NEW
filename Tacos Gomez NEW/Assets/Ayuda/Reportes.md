# 📊 Análisis de Negocio
El módulo de reportes permite conocer el estado financiero y operativo de **Tacos Gómez**.

## Cómo generar un reporte:
1. **Seleccionar Reporte:** Elige entre Ventas, Top Productos, Ranking de Empleados, etc.
2. **Definir Fechas:** Selecciona el periodo (Desde/Hasta) usando los calendarios.
3. **Analizar:** Haz clic en el botón para cargar los datos y generar las gráficas.

## 📈 Visualizaciones
- **Gráficas de Pastel:** Se muestran en reportes de Categorías y Eficiencia (Órdenes Pagadas vs Registradas).
- **Gráficas de Barras:** Se usan para los rankings de ventas y productos.

## 📁 Exportación
Puedes guardar los resultados en tu escritorio usando:
- **Botón Excel:** Crea una tabla de datos con cálculos automáticos.
- **Botón PDF:** Genera un documento listo para imprimir con el logotipo y las gráficas.
